Movie Catalog

{% comment %} Movie clipper image in the right side {% endcomment %}

I am excited to share my comprehensive Movie Catalog website developed using Next.js and TypeScript. This endeavor embodies my ongoing journey in web development, aiming to craft intuitive and functional applications for users to enjoy.

Key Features:

  • Homepage: The homepage serves as the gateway to the vast collection of movies, incorporating lazy loading for seamless browsing and a convenient search bar for effortless navigation.
  • Movie Details Page: Delve deeper into each movie's universe with detailed information, including captivating images, release dates, ratings, comments, and engaging descriptions.
  • User Accounts: Empowering users with personalized experiences, they can register securely, log in, and contribute to the community by adding ratings, comments, saving movies to their watchlist, or creating and managing movies (Editor role required).
  • Admin Panel: Administrators wield control through an intuitive interface, managing user roles effortlessly and ensuring smooth operations across the platform.
  • User Roles: From the Viewer, who can save, rate, and comment on movies, to the Editor, with the power to curate content, and the Admin, orchestrating the platform's functionality, every role adds value to the ecosystem.
  • Movie Categories: Organizing movies into distinct categories enhances discoverability, enriching the browsing experience for every cinephile.
  • Most Popular Movies: Stay updated with the pulse of the community, as the homepage showcases the week's most popular movies based on reviews and ratings.
  • Analytics Panel: Gain insights into movie popularity trends and category preferences through comprehensive analytics, with the flexibility to export data in PDF format for further analysis.
  • Film Slideshow: Immerse yourself in the cinematic realm with captivating slideshows accompanying each movie, enriching the visual experience.

Technical Implementation:

Utilizing the power of Next.js and TypeScript, this project embodies my commitment to leveraging cutting-edge technologies for seamless user experiences. With a PostgreSQL database housing the data, the project stands as a testament to robust backend architecture and meticulous frontend design.
